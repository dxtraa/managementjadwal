Setup Database Diperlukan

Database belum di-setup. Ikuti langkah-langkah berikut:

  1. Login ke Supabase Dashboard
  2. Pilih project Anda
  3. Buka SQL Editor di sidebar kiri
  4. Copy dan jalankan SQL berikut: 
    -- Buat tabel schedules
CREATE TABLE IF NOT EXISTS public.schedules (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    title text NOT NULL,
    description text,
    schedule_date date NOT NULL,
    schedule_time time NOT NULL,
    status text DEFAULT 'pending',
    created_at timestamp with time zone DEFAULT timezone('utc'::text, now())
);

-- Aktifkan RLS
ALTER TABLE public.schedules ENABLE ROW LEVEL SECURITY;

-- Buat policy untuk akses publik
CREATE POLICY "Enable all access for all users" 
ON public.schedules 
FOR ALL 
USING (true) 
WITH CHECK (true);
  5. Klik Run untuk mengeksekusi SQL
  6. Refresh halaman ini setelah setup selesai

Catatan: Untuk key yang benar, buka Settings → API di dashboard Supabase, lalu copy anon public key.
